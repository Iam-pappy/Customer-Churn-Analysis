Customer Churn Analysis and Retention Insights

📊 Project Overview
This project analyzes customer churn for a telecommunications company using the Telco Customer Churn dataset. The goal was to identify key drivers of churn and provide actionable retention recommendations.

🎯 Key Objectives
- Identify top drivers of customer churn
- Create an interactive Power BI dashboard
- Develop data-driven retention recommendations

📈 Key Findings
| Driver | High-Risk Segment | Churn Rate |
|--------|-------------------|------------|
| Contract Type | Month-to-month | 42.71% |
| Payment Method | Electronic check | 45.29% |
| Tenure | Less than 6 months | 54.27% |

📁 Project Files
| File | Description |
|------|-------------|
| `Telco_Customer_Churn.csv` | Raw dataset (7,043 records) |
| `Churn_Analysis_Pivot.xlsx` | Excel pivot tables|
| `Customer_Churn_Dashboard.pbix` | Power BI interactive dashboard |
| `Churn_Presentation_Final.pptx` | Final presentation slides |

🛠 Tools Used
- **Excel** - Pivot tables
- **Power BI** - Interactive dashboard, DAX measures
- **PowerPoint** - Presentation deck

💡 Recommendations
1. Convert month-to-month customers to annual contracts (10% discount)
2. Incentivize auto-pay enrollment ($2-3 monthly discount)
3. Implement "First 90 Days" retention program
4. Bundle security + support free for first year
